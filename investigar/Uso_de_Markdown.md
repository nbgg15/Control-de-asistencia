 ¿Qué es Markdown?

Markdown es un lenguaje de marcado ligero que permite redactar
documentos de forma simple utilizando solo texto plano. Su objetivo
principal es facilitar la creación de contenido estructurado que pueda
convertirse fácilmente en HTML u otros formatos, manteniendo una
sintaxis legible incluso sin renderizar.

¿Por qué se utiliza en proyectos de software?

-   Sintaxis sencilla y fácil de aprender.
-   Ideal para documentación técnica.
-   Compatible con GitHub, GitLab y otras plataformas.
-   Facilita el control de versiones.
-   Archivos ligeros y fáciles de editar.
